KEY FILES EXPLAINED
================================================================

BACKEND FILES:
--------------
main.py
  - Flask web server
  - API endpoints: /api/chat, /api/test-keys, /api/health
  - Integration with Google AI Studio and Cerebras Cloud
  - Supports 4 models: tutoring, academic_counseling, nutrition_health, sports
  - Bilingual support (EN/FA)
  - Automatic fallback between AI providers

requirements.txt
  - flask: Web framework
  - flask-cors: Cross-origin resource sharing
  - google-generativeai: Google AI Studio (Gemini)
  - requests: HTTP library for Cerebras API

test_api_keys.py
  - Tests Google AI Studio connection
  - Tests Cerebras Cloud connection
  - Displays results and error messages

FRONTEND FILES:
---------------
index.html
  - Single-page application structure
  - Header with logo and controls
  - 4 circular model selection buttons
  - Chat interface (hidden until model selected)
  - API status panel
  - Bilingual data attributes

style.css
  - Modern responsive design
  - CSS variables for theming (light/dark)
  - Rotating sphere logo animation
  - Golden ring light effects
  - Pulse and shine animations
  - Mobile-first responsive design
  - RTL support for Persian

app.js
  - Event handling for all interactions
  - Language toggle (EN â†” FA)
  - Theme toggle (Light â†” Dark)
  - Model selection and chat management
  - API communication with axios
  - Message rendering and animations
  - API status testing
  - LocalStorage for preferences

BATCH SCRIPTS:
--------------
TEST_INSTALLATION.bat
  - Checks Python, Node.js, npm
  - Creates virtual environment
  - Installs all dependencies
  - One-time setup

RUN_APP.bat
  - Starts backend on port 5174
  - Starts frontend on port 5173
  - Opens two terminal windows
  - Quick launch for development

NGROK_SETUP.bat
  - Creates ngrok tunnels for both services
  - Instructions for mobile configuration
  - Requires ngrok to be installed

BUILD_FOR_MOBILE.bat
  - Builds production-ready frontend
  - Outputs to dist/ folder
  - Optional preview mode

RUN_API_TEST.bat
  - Tests API connectivity
  - Verifies Google AI Studio
  - Verifies Cerebras Cloud
  - Shows detailed results

API ENDPOINTS
================================================================

GET  /api/health
     Returns: {"status": "healthy", "message": "..."}

POST /api/chat
     Body: {
       "message": "user message",
       "model": "tutoring|academic_counseling|nutrition_health|sports",
       "language": "en|fa",
       "provider": "google_ai|cerebras"
     }
     Returns: {
       "success": true/false,
       "response": "AI response",
       "provider": "provider name"
     }

GET  /api/test-keys
     Returns: {
       "google_ai": {"working": true/false, "message": "..."},
       "cerebras": {"working": true/false, "message": "..."}
     }
